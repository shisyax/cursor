# Carnitrix まとめ

Carnitrix（カーニトリックス）は、「Ben 10」をもとにしたファンメイドのホラー AU です。正式名は「Analog Ben」で、通称が「The Carnitrix」です。2023 年の最初の物語と、2026 年から続く作り直し版があります。主人公のベンは、人を助けるための時計ではなく、人を殺すための時計を左手首にはめてしまいます。

## どんな話か

ベンは Omnitrix の代わりに Carnitrix を拾います。装置は触手を伸ばして左腕に食い込み、二度と外れません。以来、近くに「脅威」を感じ取るたびに、ベンの体は本人の意思と関係なく凶暴なエイリアンへ作り変わります。意識ははっきり残ったまま、自分の手が人を殺していくのを、ただ見ているしかありません。幼いベンはこの体験で深く傷つき、やがて PTSD に近い心の傷を抱えます。

## 装置 Carnitrix

Carnitrix は、未完成のまま動き出したプロトタイプです。仕上がっていないぶん、変身のたびに耐えがたい痛みが走り、エイリアンの本能がベンをのみ込みます。理性を保てないのに、自分のしていることだけははっきり分かる——この矛盾が、何よりの恐怖です。

- 脅威を感じ取ると、Carnitrix はひとりでに起動します。ベンは変身を始めることも、途中で止めることもできません。
- 一度の変身は、およそ 20 分続きます。
- 未完成ゆえの不具合で、エイリアンはしばしば「変異」し、ベンの性別と関係なく雌雄両方の特徴を帯びます。
- 装置には Galvan の DNA だけがありません。もともと作り手の Azmuth 自身が使うはずの道具だったからです。

## 生みの親 Azmuth

この世界の Azmuth は、原作の Azmuth とは別人の、同じ名を持つ Galvan です。自分の種族以外のすべてを憎み、ほかの種族を支配して宇宙に永遠の苦しみをばらまくために Carnitrix を作りました。

けれど装置は完成しませんでした。計画を嗅ぎつけた Plumber の Vilgax と Tetrax が研究室に踏み込もうとする中、Azmuth は薬で自ら命を絶ちます。その直前、彼は Carnitrix を弱い生き物の住む星——地球——へ送り出し、一枚のメモを残します。「私が宇宙を苦しめられないなら、ほかの誰かがやる」。

## 物語の流れ

最初の犠牲者は、いとこの Gwen でした。痛みに叫ぶベンの声を聞いて駆けつけた Gwen の前で、ベンの体が炎を噴き、Heatblast に変わります。動けないベンは「逃げて」と叫びますが、間に合いません。ベンは Gwen の顔を焼き、その頭を握りつぶします。

我に返ったベンは、祖父の Max に罪を打ち明けに行きます。ところが Max を前にして装置がまた動き出し、今度は XLR8 になります。超高速で走り抜けたベンは、片脚だけを残して祖父の命を奪います。

家族を失ったベンは家をなくし、あてもなくさまよいます。Carnitrix は近づく者を脅威とみなして起動し続け、殺戮が止まりません。やがて、装置を追う Plumber の Vilgax がベンを見つけます。Vilgax は一時は優勢でしたが、最後はベンが討ち取ります。Azmuth の被造物 Malware との戦いもあり、Malware は時計から XLR8 のサンプルを一つ消しますが、ベンを倒すには至りません。

転機は、軍がベンを包囲した日でした。一度に大勢の脅威を感じ取ったベンは、最強の存在 Alien X に変わります。Alien X となったベンは、自分の宇宙そのものを消し、さらに壊すものを求めて Omniverse をさまよい始めます。

これを見た Professor Paradox は、多元宇宙のベンたちを集めます。とどめを刺したのは、原作世界の Ben 10K でした。彼は Carnitrix の脅威検知が届かない位置から Atomic-X で時計を撃ち、ベンの左腕ごと装置を壊します。それでも、ベンの命だけは奪いませんでした。

その後、Celestialsapien たちの力で、ベンの宇宙は元に戻ります。けれどベンは、装置が残した DNA の影響からか、大人になってもエイリアンや作り手の幻を見続けます。ある日、訪ねてきた Paradox に、ベンは Carnitrix をはめる直前まで時間を戻してほしいと頼みます。Paradox は、戻れば自分の存在が消えると告げたうえで、その願いを受け入れます。過去に現れた大人のベンは、幼い自分の肩に手を置いて「そのままにしておけ」と告げ、笑みを浮かべて消えます。

## Carnitrix のエイリアンたち

ベンの時計には、原作の 9 体に早くから Alien X を加えた顔ぶれが入っています。どれも本来の能力を残したまま、捕食者の本能をむき出しにした姿です。

- **Heatblast** … Pyronite のサンプルです。炎を操り、吸収します。Gwen の頭を握りつぶしたのが、この姿です。
- **XLR8** … Kineceleran のサンプルで、この AU の看板役です。時速 700km 近くで走り、ぶつかるだけで人を砕きます。狡猾でサディスティックな独自の人格があり、のちに Malware が消し去ります。
- **Four Arms** … Tetramand のサンプルです。Max と Gwen の死後、画面の外でも暴れた様子がうかがえます。
- **Ghostfreak（Zs'Skayr）** … Ectonurite のサンプルです。装置の改変が意識をのみ尽くし、もはや本来の自我は残っていません。
- **Highbreed** … Azmuth がその種族を滅ぼす直前に DNA を取り込んだサンプルです。
- **Alien X** … Celestialsapien のサンプルに Azmuth の DNA を重ねた存在です。本来そばにいるはずの Bellicus と Serena を「始末した」という設定で、その正体は Azmuth 自身ではないか、という見方もあります。

このほか Diamondhead、Wildmutt、Upgrade、Ripjaws なども、ファンや公式のリデザインがホラー寄りの姿に描き直しています。

## テーマ

Carnitrix の怖さは、力の強さより「自分の体を自分で動かせない」ことにあります。変身は新しい力を得る瞬間ではなく、皮膚や骨が作り変わり、別の生き物の本能が体を乗っ取る瞬間です。ベンは悪人ではありません。心は善良なまま、装置のままに動く体で家族や見知らぬ人を手にかけ、その記憶に苦しみ続けます。加害者でありながら被害者でもある——この救いのなさが、Carnitrix を悲劇として際立たせます。

## 名前の由来

名前はラテン語の「carnis（肉）」につながり、「carnage（虐殺）」や「carnivore（肉食）」という言葉が重なります。人を喰らい、肉を求める Alien X の描写とも響き合います。

## 広まり方

Carnitrix は、ファンアートや 3D・Roblox のモデル、解説アニメーション、ミニコミックを通じて広がりました。YouTube では Kiervo Animations の解説アニメや、comiation・JHG Comics のコミックがファンの間でよく話題に上ります。実写 VFX 動画もあり、大きな再生数を集めています。投稿者ごとに設定やデザインが少しずつ違い、同じ Carnitrix でも作品ごとに姿が変わります。

## 注意点

Carnitrix は公式設定ではありません。流血や残酷な描写を含む、大人向けのファン作品です。最初の「Analog Ben」（2023 年）と作り直し版（2026 年）でも中身が違い、装置の起源、登場するエイリアン、結末は作品ごとに変わります。
